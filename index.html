Git Command List

git init
Git 저장소 초기화
git status
Git 상태(새롭게 추가되거나, 수정된 내용) 확인
git add [file, *]
파일 버전관리 스냅샷(Snapshots) 생성
git commit -m "메모 내용"
영구적으로 버전관리 스냅샷 파일 기록
git commit -a -m "메모 내용"
스테이지 상태 건너띄고 커밋
git diff
스테이지(Staged)되지 않은 변경사항을 화면에 출력
git diff --staged
스테이지된 이후, 변경사항을 화면에 출력
git log
커밋(Commit)된 결과를 화면에 출력
git reset HEAD {file}
파일 언스테이지드(Unstaged)
git reset HEAD^
최근 커밋 취소
git checkout -- {file}
파일 이전 커밋 스냅샷 기록으로 복귀
git remote add {원격 저장소 이름} "저장소 경로"
원격 저장소 등록
git remote
원격 저장소 이름 출력
git remote -v
원격 저장소 정보 출력
git push {원격 저장소 이름} {브랜치 이름}
로컬 Git 저장소 데이터 원격 저장소로 업로드
